# vue2.x
> fis3 + vue2.x

## 构建

``` bash
# 安装fis插件
npm install -g fis-parser-babel-5.x
npm install -g fis3-hook-node_modules
npm install -g fis3-preprocessor-js-require-css
npm install -g fis3-preprocessor-js-require-file
npm install -g fis3-parser-vue-component

# 安装依赖
npm install

# 开发环境
npm run dev

# 生成环境
npm run build
```
